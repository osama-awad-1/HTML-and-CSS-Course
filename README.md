# HTML-and-CSS-Course

### 1. Portfolio Project
A unique, colorful themed website that showcases various CSS capabilities including:
*   **Fixed Navigation:** A persistent top menu with hover effects.
*   **Video Backgrounds:** Utilizing HTML5 `<video>` with autoplay, mute, and loop functionality.
*   **Image Manipulations:** Centering, CSS filters (hue-rotation/shadows), and rounded borders.
*   **Interactivity:** Image overlay fade effects and hover states for text elements.
*   **Layouts:** A custom two-column layout and a structured, styled data table.

### 2. Bootstrap Project
This repository also serves as the foundation for the upcoming Bootstrap integration, where we will transition these custom styles into the Bootstrap grid system to enhance responsiveness and component modularity.

## Technologies Used
*   **HTML5:** Semantic structure and media embedding.
*   **CSS3:** Flexbox layout, position-fixed, transitions, filters, and pseudo-classes.
